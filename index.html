Recommendation System

This is a Jupyter Notebook of Japanese media dataset. A content-based filtering method was used to make suggestions on the content to watch based on similar tags. The data needed to be preprocessed and cleaned of nulls, as well as imputated, before being analyzed for the system. Tokenization was used to make separate tag words, so that they could be vectorized. Cosine similarity was applied to the tags to find the tags nearest to each another. If given more time, an application would have been created to make the system more user friendly.
